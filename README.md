# prototypes

Lightweight HTML prototypes for product feature explorations.

Each prototype lives in its own folder with an `index.html`. The root `index.html` is a landing page that links to all of them.

## Live

https://lunasc24.github.io/prototypes/

## Adding a new prototype

1. Create a folder named with a kebab-case slug: `mkdir my-feature`
2. Add your prototype as `my-feature/index.html`
3. Add a link to it from the root `index.html`
4. Commit and push — GitHub Pages redeploys automatically
